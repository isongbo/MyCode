一.继承的方式
    公开继承  class  Cat:public Animal{  };
    私有继承  class  Cat:private Animal{  };
    保护继承  class  Cat:protected Animal{ };

二.公开继承
    在公开继承下 父类数据 到子类之后的权限变化
           父类的公开数据 到子类之后 是公开的
           父类的保护数据 到子类之后 是保护的
           父类的私有数据 到子类之后 是隐藏的
三.私有继承
    在私有继承下  父类数据 到子类之后的权限变化
           父类的公开数据  到子类之后 是私有的
           父类的保护数据  到子类之后 是私有的
           父类的私有数据  到子类之后 是隐藏的
四.保护继承
    在保护继承下  父类数据 到子类之后的权限变化        
           父类的公开数据  到子类之后 是保护的
           父类的保护数据  到子类之后 是保护的
           父类的私有数据  到子类之后 是隐藏的  
总结:所谓的继承方式  就是父类能给子类的最大访问权限
        父类的私有数据 到子类之后是隐藏的
五. 在继承中 构造和析构的调用
     5.1 构建子类对象时 一定义会调用父类的构造函数 
           再调用子类对象的构造函数。
     5.2 子类默认调用父类的无参构造函数,当然也可以使用
           初始化列表 指定调用父类的构造函数。
     5.3 析构函数的调用顺序 一定是和构造函数的调用顺序
           相反。    
六.继承中 拷贝构造函数  和 赋值运算符的调用 
    子类对象触发拷贝构造调用时 子类默认调用父类的拷贝构
          造函数。但子类一旦提供拷贝构造函数 则不再默认调用
          父类的拷贝构造函数(转而调用无参构造)
          需要指定调用父类的拷贝构造函数。
          可以通过初始化列表指定调用 父类的拷贝构造函数。  
   
   子类对象触发赋值运算符时，子类默认调用父类的赋值运算
          符函数。但子类一旦提供赋值运算符函数 则不再默认调用
          父类的赋值运算符函数。
          需要指定调用父类的赋值运算符函数。

八.多继承 
    8.1 概念
    一个类 可以有多个直接父类   子类可以继承这多个直接
              父类的数据。
    8.2 举例

   构建子类对象时 调用父类构造函数的顺序 只和继承顺序

          中共用的功能 调用会出现歧义。
